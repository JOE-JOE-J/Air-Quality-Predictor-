# 🌫️ California AQI Risk Predictor  A machine learning project that predicts whether a given day in a California county will have unsafe air quality (AQI ≥ 100) using environmental, weather, and demographic data — without relying on PM2.5.  Built for BSAN 6070 by Bhavna Sreekumar, Sarkis Shil-Gevorkyan, and Christopher Strouse.  ---  ## Overview  Most air quality prediction models rely on PM2.5 as a predictor, but PM2.5 is directly used to calculate AQI, which means including it leaks the target variable and artificially inflates model performance. This project removes PM2.5 entirely and builds a binary classifier using only independent features: ozone concentration, weather conditions, seasonal patterns, and county-level population.  Three models were trained and compared — Logistic Regression, Random Forest, and XGBoost — across 74,000+ county-day observations spanning California from 2021 to 2025. Because unsafe AQI days represent less than 1% of observations, evaluation prioritized AUC and Class 1 recall over overall accuracy.  ---  ## Data Sources  | Source | What We Used | Link | |---|---|---| | EPA Outdoor Air Quality Data | Daily AQI and ozone by California county, 2021–2025 | https://www.epa.gov/outdoor-air-quality-data/download-daily-data | | NOAA Climate Data | Daily temperature, precipitation, wind speed | https://www.ncei.noaa.gov/cdo-web/search | | U.S. Census Bureau | County-level population estimates | https://data.census.gov |  All datasets: [Google Drive Folder](https://drive.google.com/your-link-here)  ---  ## Models  | Model | Accuracy | AUC | Class 1 Recall | Notes | |---|---|---|---|---| | Logistic Regression | 0.765 | 0.820 | 0.70 | Best minority-class recall, high false positives | | XGBoost | 0.996 | 0.908 | 0.10 | Best overall AUC | | Random Forest | 0.996 | 0.783 | 0.00 | Deployed in Streamlit app | | Tuned Random Forest | 0.994 | 0.815 | 0.00 | GridSearchCV optimized |  ---  ## Key Findings  - Ozone and wind speed were the two strongest predictors of unsafe AQI after PM2.5 removal - Temperature and season contributed meaningful signal, consistent with California's wildfire-driven summer patterns - Population carried moderate importance, suggesting urbanized counties face elevated baseline risk - Removing PM2.5 caused a substantial performance drop across all models, confirming it dominates AQI but cannot be used without leaking the target  ---
